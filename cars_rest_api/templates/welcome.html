# Enpoint Method Params Description
1 api/cars GET - Displays cars saved in Data Base
2 api/cars POST car_make, model_name Checks if provided make and model exists in external DB, then saves to local DB
3 api/rate POST car_make, model_name, rating Adds rating for given car
4 api/popular GET - Shows cars order by number of rates
